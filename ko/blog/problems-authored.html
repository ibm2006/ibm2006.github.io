Problems That I Authored So Far

Tistory 원문을 옮겨 정리한 글

https://codeforces.com/blog/entry/113687, https://codeforces.com/blog/entry/137612, https://codeforces.com/blog/entry/113093, 그 외에도 수많은 "Problems That I Authored So Far"류의 포스팅을 보고 작성해보게 되었다.

백준이 서비스 종료하기 전에 출제한 문제 아카이빙을 해두는 것을 깜박하여, 일단은 기억에 의존하여 시간순으로 작성한다. 추후 백준이 문제 보기 기능을 지원하게 되면, 각 문제에 링크를 추가로 걸어둘 예정이다.

추후 출제한 문제가 추가됨에 따라 지속적으로 업데이트될 예정이다.

퀄리티 등급

  • FUCK: 쓰레기.
  • COMMON: 문제에 하자는 없지만, 필자에게 인상을 거의 주지 못한 문제
  • RARE: 얻어갈 요소가 없지 않은 문제, 그러나 "좋은 문제"라는 감상이 바로 떠오르는지는 애매한 문제
  • EPIC: 풀이가 재미있거나, 매우 인상적인 Intuition을 기반으로 만들어진 문제
  • LEGENDARY: EPIC의 2가지 조건을 동시에 만족하는 문제
  • MYTHICAL: 설명할 필요가 없을정도로 놀랍고 혁신적인 문제
