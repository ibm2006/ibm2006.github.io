Problems That I Authored So Far

Translated and reformatted from the original Tistory post.

This was written after reading this Codeforces post, this one, this one, and many other posts in the style of "Problems That I Authored So Far."

I forgot to archive the problems I authored before Baekjoon shuts down its service, so for now I am writing them chronologically from memory. If Baekjoon later supports viewing problems again, I plan to add links to each problem.

This post will keep being updated as more authored problems are added.

Quality Scale

  • FUCK: trash.
  • COMMON: a problem with no defect, but one that barely left an impression on me.
  • RARE: a problem with something to take away, but not one that immediately feels like a "good problem."
  • EPIC: a problem with an interesting solution or built from a very impressive intuition.
  • LEGENDARY: a problem satisfying both EPIC conditions at once.
  • MYTHICAL: a problem so surprising and innovative that it needs no explanation.
